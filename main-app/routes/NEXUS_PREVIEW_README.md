## Purpose

This landing page (`/nexus-preview`) demonstrates comprehensive understanding of:

## Design Philosophy Implementation

### 1. Material Truth (Anti-Plastic Doctrine)

✅ **Implemented:**
- Surfaces use Obsidian (`#161618`) for weight and depth
- Parchment (`#E9E3D2`) for knowledge surfaces
- No pure white (`#FFFFFF`) - FORBIDDEN
- Material colors represent states, not UI states

### 2. Typography Doctrine

✅ **Editorial Authority:**
- Headings use `Cormorant Garamond` (serif) - "written, not rendered"
- High contrast strokes, generous spacing
- Letter spacing: `-0.02em` for tight, authoritative feel

✅ **Forensic Data:**
- Numeric data uses `JetBrains Mono` (monospace)
- Eliminates ambiguity, signals permanence
- Used for: "Rules: 1,247", "Latency: <200ms", "Response: <350ms"

### 3. Light as Language

✅ **Conditional Illumination:**
- Cards: Dim at rest (`border-[#2A2A2C]`), light descends on hover (`border-gold`)
- Transitions: `1000ms cubic-bezier(0.4, 0, 0.2, 1)` - gravitational time
- Borders warm before content (hover effect)

✅ **Motion Physics:**
- No bounce, snap, or elastic easing
- Hover: `1000ms` (illumination)
- Commitment: `1618ms` (Golden Ratio timing for buttons)

### 4. Interaction Law

✅ **Cards (Illumination):**
- Three knowledge cards demonstrating "The Three Domains"
- Dim at rest, border warms to gold on hover
- Background lightens subtly (`#1E1E20` on hover)
- Transition duration: `1000ms` (illumination timing)

✅ **Buttons (Ratification):**
- Quiet visibility at rest (transparent, gold border)
- Resistance before activation (`1618ms` transition)
- Time-based confirmation (Golden Ratio timing)
- Monospace font (`JetBrains Mono`)
- Letter spacing: `0.1em` (decree tracking)

### 5. Color System (Material-Based)

✅ **Core Palette:**
- **Void** (`#0E0E0F`) - Authority text
- **Obsidian** (`#161618`) - Weight surfaces
- **Parchment** (`#E9E3D2`) - Knowledge surfaces
- **Ash** (`#CFC8B8`) - Commentary text
- **Gold** (`#B8A56A`) - Ratified Authority
- **Ember** (`#8C6A2F`) - Consequence (not used in preview, but available)

### 6. Restraint & Authority

✅ **Design Principles:**
- Minimal color palette (NO color bloat)
- Generous spacing (authority through space)
- System waits (no aggressive animations)
- Interface resists (buttons require commitment)
- Interface remembers (consistent material behavior)
